O que são pilhas?

  • São estruturas de dados do tipo LIFO (last-in first-out), onde o último elemento a ser inserido, será o primeiro a ser retirado.
  • Assim, uma pilha permite acesso a apenas um item de dados - o último inserido.
  • Para processar o penúltimo item inserido, deve-se remover o último primeiro.

Seguindo essa ideia, desenvolvemos o exemplo acima, que demonstra como a estrutura de dados funciona.
